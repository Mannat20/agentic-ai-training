## Key Concepts Learned

### Object
An entity that has:
- Properties (Attributes)
- Behaviors (Methods)

### Class
A blueprint or textual representation of an object.

### Mapping
Represents relationships between objects:
- One-to-One
- One-to-Many
- Many-to-Many

---

## Memory Understanding

- Variables are stored in the Stack.
- Objects are stored in the Heap.
- Stack variables store references (addresses) of objects.
