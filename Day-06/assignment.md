# Day 06 Assignment

## Objective

Practice writing reusable functions and understand how Python handles objects in memory.

---

## Topics Practiced

- Creating Functions
- Parameters
- Default Parameters
- Reference Copy
- Mutable vs Immutable Objects
- Memory Representation
- Function Redefinition
- Variable Arguments (`*args`)
- Keyword Arguments (`**kwargs`)
- Deep Copy
- Return Statement
- Debugging using Breakpoints and Logs

---

## Practice Programs

- Function to calculate the square of a number.
- Function with default parameter values.
- Program demonstrating mutable and immutable objects.
- Program using `*args`.
- Program using `**kwargs`.
- Demonstration of deep copy.
- Function returning multiple values.
- Debugging programs using breakpoints and logs.

---

## Expected Learning

- Write modular and reusable programs.
- Understand Python's memory model.
- Pass flexible arguments to functions.
- Prevent unintended side effects using deep copy.
- Debug programs effectively using breakpoints and logs.

---

## Outcome

Successfully implemented various function-based programs while understanding debugging techniques, memory representation, and Python's object reference behavior.
